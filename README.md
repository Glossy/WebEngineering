## Task7_Web应用构建
- [Task7_Web应用构建](https://github.com/Glossy/WebEngineering)
- 选择适当的技术，完成Web应用（部分功能）构建。
- 至少有5个页面, 包括主页, 尽量让每个页面有不同点。
- 部分功能要包含架构的各种组件。如：架构中包含数据库，就需要实现访问数据库存储的功能。
- Web应用部署
- 在Web应用构建过程中，选择适当粒度进行部署。最低要求：上述构建内容部署可用。
